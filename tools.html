Copy Local Relations

Copies the local relation using the first selected object as the driver for the others.

  • Right-click to toggle realtime drag.
  • Pairs with Paste/Context for quick transfers.

World Space

Creates a world-space copy that frees a control from local rig space while keeping its motion intact.

  • Ideal for walk cycles and similar loops.
  • Right-click for bake type and axis options.

Aim Space

Creates an aim space that keeps orientation toward a dynamic point.

  • Great for looks, aiming props, or look-at setups.
  • Pick the axis from the menu and choose your bake mode.

Motion Trail

Generates a motion trail based on speed or frequency with adjustable thickness, color, and sensitivity.

  • Right-click for trail styling and markers.

Pivot Tool

Moves or rotates a temporary pivot on one or more controls without touching the rig.

  • Stores the last pivot position per control.
  • Right-click to open the mode popup and reposition it.

Euler Filter

Applies Euler filter helpers and lets you assign a custom hotkey.

  • Right-click for channel and key range options.

Bake

Bakes using the first selected object as the driver for the second (driven), works with components too.

  • Select the driver first, then the driven.
  • Right-click for more options.

Extra Features

Blend Neighbor

Blends toward the previous or next value for the selected keys.

Move Relative

Moves relatively toward the previous or next value for the selected keys.

Tween

Interpolates between the previous and next keys.

Blend Ease

Blends toward the previous or next value with an ease-style falloff for the selected keys.

Push / Pull

Pushes or pulls the values toward the interpolation between the previous and next keys.

Time Offset

Offsets the curve left or right by adjusting the key values without moving them in time.

Smooth / Rough

Pulls adjacent keys closer together or farther apart. Smooth helps remove noise like mocap.
